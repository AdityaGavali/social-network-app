SocialXpress: Social Media Application 
</br>
Tech Stack: React.js, Firebase Authentication, Firebase Database, Tailwind CSS, Redux.js

Functionalities: 
1. Registration
2. Authentication
3. Creating a post with an image and description
4. Responsive 
5. Like and comment feature
6. Friendly UI/UX
